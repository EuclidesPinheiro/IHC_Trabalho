Os Frameworks de Javascript

Com o objetivo de permitir a Reutilização de Código e garantir um Ganho de Tempo, os Frameworks são uma ótima ferramenta para os desenvolvedores modernos. Frameworks podem ser entendidos como uma abstração que combina Trechos de Código para serem Reutilizados, permitindo assim que os desenvolvedores possam focar em Resolver os Problemas ao invés de ficarem rescrevendo linhas de código.

Nesse trabalho, buscamos apresentar os principais frameworks e bibliotecas utilizadas pela Linguagem JavasScript.
