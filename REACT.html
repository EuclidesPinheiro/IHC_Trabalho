React

O React é uma biblioteca front-end e tem como um de seus objetivos facilitar a conexão entre diferentes partes de uma página, portanto seu funcionamento acontece através do que chamamos de componentes. Essa biblioteca divide uma tela em diversos componentes para, então, trabalhar sobre eles de maneira individual.

React é uma biblioteca JavaScript declarativa, eficiente e flexível para construir interfaces de usuário. É a biblioteca front-end de JavaScript mais popular do mercado para o desenvolvimento de interfaces de usuário (UI).

Criado em 2011 pelo time do Facebook, o React surgiu com o objetivo de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social, entre eles chat, status, listagem de contatos e outros.

React utiliza uma extensão alternativa ao JavaScript para descrever seus componentes. Esta sintaxe é chamada de JSX. O JSX, a grosso modo, é uma linguagem desenvolvida como uma mistura de HTML e JS. Seu objetivo é ser transposta para JS da maneira mais simplificada possível.
